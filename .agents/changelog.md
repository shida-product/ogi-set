# プロジェクト作業履歴 (Changelog)

主要な変更の判断背景を記録します。詳細な差分は Git commit を追ってください。

## [2026-07-02] トラネキサム酸250mgをトラネキサム酸500mgを含む3種セットの代替として適用可能に拡張

- 3種セットの判定ロジックを拡張し、トラネ500mg「YD」を含むセット定義において、トラネ250mg「YD」が選択されている場合も代替してセット化できるように変更。
- トラネ250mgが直接選ばれている場合、250mg×2のチェックボックス（`trane2x2`）がONであっても +¥500 の価格加算は適用しないように処理。

## [2026-07-02] スプレッドシート連携機能（GASサイドバー対応）の実装とフォールバック機能の追加

- スプレッドシートをデータソースとし、サイドバーをUIとして動作させるためのGASスクリプト `Code.js` を新規作成。
- スプレッドシートの1枚のシート「マスタ」から、縦並び（マージン見出し付き）のレイアウトで商品・セット定義を動的に読み込むように実装。商品マスタ定義からグループ（カテゴリ）列を廃止して簡素化。
- スプレッドシートのカスタムメニューからワンクリックで、簡素化・縦並びされた「マスタ」シートと初期データを自動生成・装飾する全自動セットアップ機能 (`setupMasterSheet`) を `Code.js` に追加。
- clasp（Google Apps Script CLI）接続を設定し、ローカルから直接GASプロジェクトへデプロイ可能な環境 (.clasp.json, .claspignore, appsscript.json) を構築。node_modules 等の不要なライブラリ群がプッシュされないよう .claspignore の除外指定を最適化。
- スプレッドシートのメニューに「大画面ダイアログで起動」オプションを追加し、任意の広さ（横680px×縦850px）のモードレスダイアログとしてアプリを表示できる機能を追加。
- アプリ画面（HTML）のヘッダーに「🔄 再読込」ボタンを追加。スプレッドシートの価格変更後、画面を開き直さずにワンクリックでシートから最新データを再ロードして画面に反映できる同期機能を実装。（非GASの通常ブラウザ環境下では自動的に非表示になります。）
- `index.html` および `set-price-optimizer.html` にて、GAS環境下（スプレッドシート連携時）ではスプレッドシートからデータを取得する処理を実装し、取得中はローディングを表示。
- 非GAS環境下（通常のローカル起動やGitHub Pages）では、従来通りファイル内の `DEFAULTS` を使用して動作するフォールバック設計を導入。既存の公開環境を壊さずにスプレッドシート内アプリ化を実現。

## [2026-06-28] オプション配置の最適化、3種適用、会計パネルの画面下部固定、および動的余白調整の適用

- 「トラネキサム酸を 250mg×2 で用意する」のオプションチェックボックスを、「トラネ500mg「YD」」の選択行の直下にマージン付きで動的配置するようにし、関連性を明確化。
- トラネ500mg「YD」を含む3種セットに対しても、オプションONのときに一律 +¥500 の加算と明細注記が適用されるようにロジックを拡張。
- オプション注釈（説明文）に、適用対象となる具体的なセット構成（2種・3種）を一覧で追記して分かりやすさを向上。
- 合計金額だけでなく、適用された内訳（対象セット）も含めた「会計パネル（`.result`）」全体を画面最下部に固定表示。未選択時は非表示。
- 会計パネル内での内訳スクロールを廃止し、全行が上に伸びて一覧表示されるように変更。
- 会計パネルが上に伸びた際に「ノイロビタン配合錠」やフッター注釈が下に隠れないよう、パネルの実際の高さを検知して全体の余白（`padding-bottom`）をJavaScriptで動的かつ自動的に調整するロジックを追加。

## [2026-06-28] 文言修正・改行の適用、および数量ボタンの拡大

- トラネキサム酸250mg×2の調剤ラベル文言の「調剤する」を「用意する」に変更。
- トラネキサム酸250mg×2の注釈文およびフッターの「仕組み」説明文を、句点などの適切な位置で改行（`<br>`）するように調整。
- 数量増減ボタン（`.qbtn`）のサイズを従来の1.3倍（幅42px・高さ42px、フォントサイズ22px、角丸10px）に拡大し、操作性を向上。

## [2026-06-28] グループタイトルの排除

- 画面上に表示されていた「主剤（C・システイン）」、「肝斑（トラネキサム）系」、「ビタミンE系」、「その他」などのグループタイトル（`.group-label`）の表示を全て排除（非表示化）。
- 薬リストはそのまま表示し、余計な見出し表示を減らすことで視認性を高めた。
