# 編集ロック（掲示板）

> 技術的な排他制御ではない。**誰がどのファイルを触っているか**を全セッションで共有する掲示板。
> ロック中のファイルは編集しない。**それ以外のファイルは並列作業してよい。**

## ルール

1. **編集を始める前**に、触る予定のファイルをこの表に追記する。
2. **他セッションがロック中のファイルは編集しない**（別ファイルへの切り替え、待機、handover 更新は可）。
3. **作業区切り**（`session-close.md`）で自分の行を削除する。
4. 30 分以上更新がない行は stale とみなし、次に作業するセッションが削除してよい。

## セッション ID の付け方

一意になればよい。例:

- `cursor-20260605-auth`
- `claude-20260605-refactor`
- `antigravity-20260605-fix`

## アクティブロック

| セッション ID | ロックファイル | 目的（1行） | 更新時刻 |
|---|---|---|---|
| （作業開始時に追記） | | | |

## 共有ファイルの扱い

`.agents/handover.md` / `.agents/state/locks.md` / `.agents/RULES.md` などは:

- **読む**: いつでも可
- **書く**: 自分の作業に必要なとき。同時編集の衝突が起きたら、先に書いた内容を尊重し差分を統合する
